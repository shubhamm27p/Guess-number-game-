# Guess-number-game

🔢 Guess the Number Game
A simple and fun number guessing game built using HTML and JavaScript. Test your luck and logic by trying to guess the secret number!

✨ Features
🎮 Interactive gameplay with instant feedback

🔄 Restart option to play multiple rounds

📊 Score tracking to challenge yourself

💻 Lightweight and works in any browser

🛠️ Tech Stack
HTML5 – game structure and layout

CSS3 – styling and basic design (optional if you added styles)

JavaScript (ES6) – game logic, random number generation, and user interaction

🚀 How to Play
Enter a number within the given range.

Click Submit to check your guess.

Get feedback:

✅ Correct guess → You win!

🔼 Too high → Try a smaller number.

🔽 Too low → Try a bigger number.

Keep guessing until you find the secret number.

📂 Project Goals
This project was created to practice JavaScript fundamentals such as random number generation, conditional statements, and DOM manipulation.
